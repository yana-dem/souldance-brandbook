# Иконки и декоративные элементы — Soul Dance

---

## Иконки — единый набор для сайта и материалов

### Выбранный набор: Phosphor Icons

Сайт: https://phosphoricons.com
Подключение: через CDN или SVG-спрайт

**Почему Phosphor:**
- Стиль «regular» — средняя линия, хорошо читается на тёмном фоне
- Большой выбор, включая иконки соцсетей
- Бесплатный, открытый

### Стиль — только «regular»

```
✓ regular     — для всего сайта и материалов
✗ thin        — теряется на тёмном фоне
✗ bold        — слишком грубо рядом с округлым логотипом
✗ fill        — только для активного состояния
✗ duotone     — не использовать
```

### Размеры иконок

```
16px — внутри текста, мелкие метки
20px — интерфейсные элементы, кнопки
24px — основной размер в карточках и списках
32px — акцентные иконки в секциях
48px — крупные одиночные иконки
```

### Цвета иконок

```css
color: var(--color-text);    /* #f0ecea — основная */
color: var(--color-muted);   /* #9e8f8f — второстепенная */
color: var(--color-accent);  /* #febad0 — акцентная */
color: #0d0d0d;              /* на розовом фоне кнопки */
```

### Иконки соцсетей

Из набора Phosphor: TelegramLogo, VkLogo, YoutubeLogo, TiktokLogo.
Для MAX — официальный SVG-логотип MAX.

### Правила

✓ Все иконки из одного набора Phosphor, один стиль
✓ Иконка всегда сопровождается текстом или явным контекстом
✓ Размер из шкалы выше
✗ Смешивать с другими наборами на одной странице
✗ Эмодзи вместо иконок (🚀 ⚡ 🎯) — запрещено на сайте
✗ Иконки разных стилей вперемешку

## Декоративная система — пять элементов

Все декоративные элементы Soul Dance — из визуального языка бренда.
Ничего нового не придумываем, работаем только с этими пятью.

---

### Элемент 1 — Звёзды логотипа (пятиконечные, скруглённые)

Из правой части логотипа — три звезды разного размера рядом с кометой.
Форма мягкая, скруглённая, в духе логотипа.

**Когда:** разделители, декор угла поста, фоновый паттерн, рядом с заголовком.

```
Цвет:    #febad0 (основной) / #2a1a1f (тихий фоновый)
Размеры: группа из 2–3 штук — крупная / средняя / мелкая
Угол:    варьировать в диапазоне ±15°
```

✗ Одна звезда отдельно — не работает, только группой
✗ Равномерная сетка из одинаковых звёзд

---

### Элемент 2 — Sparkle-звёздочки (четырёхлучевые)

Четыре луча, вытянутые крест-накрест — как в посте «Важные обновления».
Более острые и динамичные чем звёзды логотипа.

**Когда:** акцент рядом с заголовком, мелкий декор рядом с текстом,
группа из 2–3 штук разного размера для оживления пустого пространства.

```
Цвет:    #f0ecea (белый) или #febad0
Размеры: обычно мелкие — 12 / 16 / 24px в группе
Сочетание: хорошо работают вместе со звёздами логотипа
```

✗ Крупные sparkle поверх текста — мешают чтению
✗ Больше четырёх штук на одном макете

---

### Элемент 3 — Клякса-звезда / Blob star (четырёхконечная)

Пухлая четырёхконечная форма — как на сертификате слева.
Мягкая, округлая, органическая. Занимает больше места чем sparkle.

**Когда:** крупный акцентный декор на макете, элемент в углу карточки
или сертификата, замена иллюстрации в пустой зоне.

```
Цвет:    #febad0 / #f0ecea / #2a1a1f (фоновая версия)
Размер:  от 40px — мельче теряет форму
Сочетание: хорошо рядом с огоньком или звёздами логотипа
```

✗ Несколько blob star разного размера на одном макете — хаос
✗ Использовать как иконку (слишком абстрактно без контекста)

---

### Элемент 4 — Волнистые линии

Плавные изогнутые линии по краям макета — как в посте «Важные обновления».
Создают ощущение движения и энергии, обрамляют контент.

**Когда:** фоновый декор поста или сторис, обрамление по углам,
разделитель между блоками в горизонтальном варианте.

```
Цвет:    #febad0 с opacity 0.3–0.5 / тёмно-розовый #3d1f28
Толщина: 3–6px, не тоньше — теряется на тёмном фоне
Позиция: всегда по краям, не поверх текста
```

✗ Волны поверх лиц или важного текста
✗ Яркие волны на весь макет — становятся фоном, теряют смысл
✗ Слишком острые углы — форма должна быть плавной

---

### Элемент 5 — Диагональный паттерн / штрихи

Полосы под углом ~45° — как фоновая текстура в прайсе и на сертификате.
Добавляет глубину и фактуру без отвлечения от контента.

**Когда:** фоновая текстура тёмных карточек, подложка под прайс,
фон обложки YouTube или анонса.

```
Цвет:    чуть светлее фона — #161616 на #0d0d0d / или #febad0 opacity 0.05–0.1
Угол:    45° или -45° — строго, не произвольно
Толщина: тонкие (1–2px) и расстояние между полосами 12–20px
```

✗ Контрастный паттерн — должен быть едва заметным
✗ Другой угол кроме 45° / -45°
✗ Поверх текста без затемняющей подложки

## Как сочетать элементы

На одном макете — не больше двух-трёх элементов из пяти.
Слишком много декора = визуальный шум.

**Рабочие сочетания (проверено на реальных постах):**

| Тип контента | Элементы |
|---|---|
| Анонс курса / события | Волны по краям + sparkle у заголовка |
| Прайс / информация | Диагональный паттерн + звёзды логотипа внизу |
| Сертификат / подарок | Blob star + sparkle + диагональный паттерн |
| Обновления / новости | Волны + sparkle (как в реальном посте) |
| Видео кавера | Только звёздный фон — тихо, чтобы не отвлекать |
| Праздничный пост | Звёзды логотипа + sparkle + волны |

**Чистый тёмный (#0d0d0d без декора):**
Для серьёзного контента — важные объявления, юридическая информация,
посты где главное — текст или лицо человека.

---

## Цветочки и другие случайные элементы

Цветочки (как в посте «Летний к-поп день») — разовое оформление
для конкретного события. В системный декор не входят.

Правило: если элемент не из пяти описанных выше — он разовый
и не должен повторяться в других материалах как системный.

## Полупрозрачные элементы — фирменная деталь Soul Dance

Яна особо отмечает полупрозрачные формы как любимый визуальный приём.
Это работает и в видео, и в статичных макетах.

### Виньетка

Полупрозрачное свечение по краям кадра или изображения.
Создаёт глубину и атмосферу — особенно в светлом зале.

```
Цвет:       #febad0 (розовая) или #0d0d0d (тёмная)
Форма:      размытый круг / овал по краям
Прозрачность: 20–40% — едва заметно, но работает
Где:        видео, постановочные фото, hero-секция сайта
```

### Полупрозрачные круги и пятна

Размытые розовые круги на фоне — не чёткие формы, а мягкое свечение.
Добавляются в монтаже или в графических редакторах.

```
Цвет:       #febad0 с opacity 15–35%
Blur:       сильное размытие — форма угадывается, не читается чётко
Где:        фон видео, углы постов, фоновый слой hero-секции
```

### Полупрозрачная подложка под текст

Розовый или тёмный прямоугольник под заголовком — вместо обводки.
Читается чище и современнее.

```css
/* Розовая подложка */
background: rgba(254, 186, 208, 0.45);
border-radius: 4px;
padding: 4px 12px;

/* Тёмная подложка */
background: rgba(13, 13, 13, 0.65);
border-radius: 4px;
padding: 4px 12px;
```

В видео (CapCut): цветной прямоугольник на слой выше видео,
прозрачность 40–60%, поверх него текст белый Russo One.

### Правила использования

✓ Полупрозрачные элементы — всегда розовые (#febad0) или тёмные (#0d0d0d)
✓ Размытие обязательно — чёткий цветной круг выглядит иначе
✓ Можно накладывать на фото и видео — это и есть смысл приёма
✗ Не использовать другие цвета для полупрозрачных форм
✗ Не делать слишком ярко — эффект тонкий, не кричащий

- [ ] Иконки из Phosphor, стиль regular
- [ ] Декор только из пяти системных элементов
- [ ] Не больше двух-трёх декоративных элементов на макет
- [ ] Декор не перекрывает текст и лица
- [ ] Фон #0d0d0d — не белый и не случайный
- [ ] Если добавлен новый элемент — это осознанное решение, не «попалось»

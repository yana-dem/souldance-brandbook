# Принципы хорошего дизайна — Soul Dance

Эти правила применяются при создании любой страницы или компонента.
Источник: опыт senior UI/UX дизайнеров, адаптирован под Soul Dance.

---

## Иерархия — главный закон

- Один экран — одна главная мысль, одно главное действие
- Размер + жирность + цвет работают вместе: заголовок крупный, подзаголовок тише, кнопка — единственное розовое пятно
- Акцентный цвет (`#febad0`) занимает максимум 5–10% площади экрана
- **Тест прищуром:** расфокусируй взгляд на макете — главная кнопка должна выделяться первой

---

## Сетка и воздух

- Связанные элементы стоят ближе друг к другу, чем несвязанные
- Между секциями страницы: 96–128px (десктоп), 48–64px (мобайл)
- Внутри карточки: 24–32px
- Когда сомневаешься — добавь воздуха, а не рамку
- Текст не растягивается на всю ширину монитора никогда

---

## Мобильная версия — сначала

1. Собирать страницу с 375px
2. Потом расширять до 768px и 1280px
3. Шире 1440px — контент не растягивается, центрируется

**Мобильные правила:**
- Все колонки складываются в одну
- Кнопки — на всю ширину, высота от 48px
- Боковые поля: 16px
- Таблицы → карточки или списки
- Меню → гамбургер справа, логотип слева
- Картинки — под текст, на всю ширину

---

## Компоненты — все состояния обязательны

Каждый интерактивный элемент имеет:
- `default` — обычное
- `hover` — наведение (150ms ease)
- `active` — нажатие (scale 0.98)
- `focus-visible` — фокус клавиатуры (outline 2px accent)
- `disabled` — недоступное (opacity 0.4)

Кнопка без hover — ощущается сломанной.
Зона нажатия — минимум 44×44px.

---

## Ритм блоков — не три карточки везде

❌ Признак AI-генерации:
```
Заголовок по центру
Подзаголовок-вода по центру
[Карточка] [Карточка] [Карточка]

Заголовок по центру
Подзаголовок-вода по центру
[Карточка] [Карточка] [Карточка]
```

✓ Правильный ритм — чередовать:
- Карточки в сетке
- Текст + изображение (слева/справа)
- Цитата-отзыв на всю ширину
- Цифры/статистика горизонтально
- Снова карточки с другим размером

Заголовки секций — не всегда по центру. Иногда слева — это живее.

---

## Тени — минимально

- Максимум 2 уровня «высоты» на всей странице
- Тени мягкие: `box-shadow: 0 4px 24px rgba(0,0,0,0.3)`
- Жёсткая чёрная тень под каждой карточкой — дешёвый дизайн
- В тёмной теме тени почти не работают — используем осветление поверхности

---

## Анимации — служебные

- Длительность: 150–300ms
- Назначение: подчеркнуть действие, не развлекать
- Ничего не мигает и не крутится само по себе
- `prefers-reduced-motion`: все анимации отключаются

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## Запрещённые паттерны — маркеры AI-генерации

❌ **Цвета:**
- Фиолетово-синие градиенты
- Градиент на каждом заголовке и кнопке
- Чисто-белый #FFFFFF + чисто-чёрный #000000
- Стеклянные полупрозрачные карточки без причины
- Неоновое свечение вокруг элементов

❌ **Тексты:**
- «Раскройте свой потенциал», «Инновационные решения», «Выведите на новый уровень»
- Рубленые фразы через точку: «Без воды. Только практика. Результат.»
- Идеально симметричные списки по 3 слова в каждом пункте
- Кнопки «Узнать больше» и «Начать сейчас» без объекта

❌ **Данные:**
- Выдуманные цифры (97% клиентов довольны)
- Несуществующие отзывы
- Одинаковые круглые цифры везде: 100+, 500+, 1000+
- Если данных нет → `[УТОЧНИТЬ У ЯНЫ]`, не придумывать

❌ **Изображения:**
- Эмодзи вместо иконок (🚀 ⚡ 🎯)
- Сток-фото рукопожатий и офисов
- Иконки разных стилей вперемешку

❌ **Компоненты:**
- Кнопка без состояний hover/active/focus
- Форма без обработки ошибок и подтверждения отправки
- Счётчики обратного отсчёта, сбрасывающиеся при перезагрузке
- Ссылки ведущие в `#` (никуда)

## Чек-лист перед сдачей любой страницы

- [ ] Одна главная мысль на первый экран
- [ ] Все отступы из шкалы (4/8/12/16/24/32/48/64/96/128px)
- [ ] Все интерактивные элементы имеют все состояния
- [ ] Контраст текста: основной 4.5:1+, крупные заголовки 3:1+
- [ ] Зоны нажатия минимум 44×44px
- [ ] Нет горизонтального скролла на 375px
- [ ] Весь текст читается без увеличения на мобайле
- [ ] Нет выдуманных данных (всё `[УТОЧНИТЬ]` или реальное)
- [ ] Нет стоп-слов (урок / класс / ученик / пробное занятие)
- [ ] Нет эмодзи
- [ ] Кавычки «ёлочки», тире длинное
- [ ] Формы: три чекбокса, не отмечены по умолчанию
- [ ] Нет маркеров AI-генерации из списка выше

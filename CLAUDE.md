## Проект

Soul Dance — к-поп танцевальная студия в Москве.
Сайт: GitHub Pages → домен souldance.ru (подключается позже).
Правовой статус: **развлекательная площадка**, не образовательное учреждение.

## Стоп-слова — НИКОГДА не использовать

| ❌ Запрещено | ✅ Правильно |
|---|---|
| урок | занятие |
| класс | занятие |
| ученик | участник |
| пробное занятие | первое занятие |
| преподаватель | хореограф |

## Цвета — только эти

```
--color-bg:        #0d0d0d   /* фон страницы */
--color-surface-1: #161616   /* карточки */
--color-surface-2: #1f1f1f   /* теги, плашки */
--color-text:      #f0ecea   /* основной текст */
--color-muted:     #9e8f8f   /* подписи, мета */
--color-accent:    #febad0   /* ЕДИНСТВЕННЫЙ акцент */
--color-accent-hover: #e8a0b8
--color-accent-soft:  #2a1a1f
--color-border:    #2e2020
--color-success:   #4caf7d
--color-error:     #c0544a
```

Второй акцентный цвет — запрещён. Фиолетовые/синие градиенты — запрещены.

## Шрифты

- Заголовки, кнопки, теги: **Russo One** (Google Fonts, кириллица родная)
- Текст, интерфейс, подписи: **Montserrat** (Google Fonts)

Unbounded — не использовать: кириллица не родная, русские буквы выглядят скованно.

## Правила всегда

- Отступы только из шкалы: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128 px
- Кнопки = глагол + объект: «Записаться на интенсив», не «Отправить»
- Кавычки «ёлочки», тире — длинное (—)
- Эмодзи на сайте — запрещены
- Выдуманные данные — запрещены; если данных нет → пишем [УТОЧНИТЬ]
- Mobile first: сначала 375px, потом десктоп
